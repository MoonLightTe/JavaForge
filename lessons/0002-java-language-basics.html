第一部分：Java vs JavaScript 核心差异

1.1 类型系统

JavaScript（动态类型）
let name = "张三"; // string let age = 25; // number let isStudent = true; // boolean // 类型可以随时改变 name = 123; // ✅ 合法
Java（静态类型）
String name = "张三"; // 必须声明类型 int age = 25; boolean isStudent = true; // 类型不可改变 name = 123; // ❌ 编译错误

1.2 基本数据类型

Java 类型 大小 范围 JavaScript 等价
byte1 字节-128 ~ 127Number
short2 字节-32768 ~ 32767Number
int4 字节-2^31 ~ 2^31-1Number
long8 字节-2^63 ~ 2^63-1BigInt
float4 字节单精度浮点Number
double8 字节双精度浮点Number
char2 字节Unicode 字符String
boolean1 位true/falseBoolean
// Java 变量声明 int age = 25; double price = 99.99; char grade = 'A'; boolean isActive = true; String name = "张三"; // String 是类，不是基本类型

1.3 类型转换

// 自动类型转换（小 → 大） int myInt = 100; double myDouble = myInt; // ✅ 自动转换 // 强制类型转换（大 → 小） double pi = 3.14; int rounded = (int) pi; // ⚠️ 丢失小数部分，结果为 3

第二部分：控制流语句

2.1 条件语句

JavaScript
if (score >= 90) { console.log("优秀"); } else if (score >= 60) { console.log("及格"); } else { console.log("不及格"); } // 三元运算符 let result = score >= 60 ? "Pass" : "Fail";
Java
if (score >= 90) { System.out.println("优秀"); } else if (score >= 60) { System.out.println("及格"); } else { System.out.println("不及格"); } // 三元运算符（相同） String result = score >= 60 ? "Pass" : "Fail";

2.2 循环语句

// for 循环 for (int i = 0; i < 10; i++) { System.out.println(i); } // while 循环 int count = 0; while (count < 5) { System.out.println(count); count++; } // 增强 for 循环（for-each） int[] numbers = {1, 2, 3, 4, 5}; for (int num : numbers) { System.out.println(num); }

2.3 switch 语句

switch (day) { case "Monday": System.out.println("周一"); break; case "Tuesday": System.out.println("周二"); break; default: System.out.println("其他天"); }

第三部分：面向对象编程

3.1 类和对象

核心概念：
  • 类（Class）：对象的模板，定义属性和方法
  • 对象（Object）：类的实例
  • 属性（Field）：对象的状态
  • 方法（Method）：对象的行为
// 定义类 public class Person { // 属性 private String name; private int age; // 构造方法 public Person(String name, int age) { this.name = name; this.age = age; } // Getter 方法 public String getName() { return name; } // 方法 public void sayHello() { System.out.println("你好，我是 " + name); } } // 创建对象 Person person = new Person("张三", 25); person.sayHello(); // 输出：你好，我是 张三

3.2 封装

封装：将数据（属性）和操作数据的方法捆绑在一起，并隐藏内部实现细节。
public class BankAccount { private double balance; // 私有属性 // 通过公共方法访问 public double getBalance() { return balance; } // 添加验证逻辑 public void deposit(double amount) { if (amount > 0) { balance += amount; } } }

3.3 继承

// 父类 public class Animal { protected String name; public void eat() { System.out.println(name + "正在吃东西"); } } // 子类继承父类 public class Dog extends Animal { public Dog(String name) { this.name = name; } // 子类特有方法 public void bark() { System.out.println(name + "汪汪叫"); } } Dog dog = new Dog("旺财"); dog.eat(); // 继承自 Animal dog.bark(); // Dog 特有

3.4 多态

// 方法重写 public class Cat extends Animal { @Override public void eat() { System.out.println(name + "优雅地吃猫粮"); } } // 多态：同一方法，不同行为 Animal animal1 = new Dog("旺财"); Animal animal2 = new Cat("咪咪"); animal1.eat(); // 旺财正在吃东西 animal2.eat(); // 咪咪优雅地吃猫粮

第四部分：接口和抽象类

4.1 接口（Interface）

接口：定义一组方法签名，实现类必须实现这些方法。
// 定义接口 public interface Drawable { void draw(); // 抽象方法 } // 实现接口 public class Circle implements Drawable { @Override public void draw() { System.out.println("画一个圆"); } }

4.2 抽象类（Abstract Class）

public abstract class Shape { protected String color; // 抽象方法（子类必须实现） public abstract double getArea(); // 普通方法 public void display() { System.out.println("面积：" + getArea()); } }

第五部分：异常处理

5.1 异常层次结构

Throwable
├── Error（严重错误，不应捕获）
│   ├── OutOfMemoryError
│   └── StackOverflowError
└── Exception（可捕获异常）
    ├── 受检异常（必须处理）
    │   ├── IOException
    │   └── SQLException
    └── RuntimeException（非受检异常）
        ├── NullPointerException
        ├── ArrayIndexOutOfBoundsException
        └── ClassCastException

5.2 try-catch-finally

try { // 可能抛出异常的代码 int result = 10 / 0; } catch (ArithmeticException e) { // 捕获并处理异常 System.out.println("除零错误：" + e.getMessage()); } catch (Exception e) { // 捕获其他异常 System.out.println("发生错误：" + e.getMessage()); } finally { // 无论是否异常都会执行 System.out.println("清理资源"); }

5.3 throws 和 throw

// throws：声明方法可能抛出的异常 public void readFile(String path) throws IOException { // 可能抛出 IOException } // throw：手动抛出异常 public void setAge(int age) { if (age < 0) { throw new IllegalArgumentException("年龄不能为负数"); } this.age = age; }

6.1 数组

// 声明和初始化 int[] numbers = new int[5]; // 指定大小 int[] scores = {90, 85, 95, 88, 92}; // 直接初始化 // 访问元素 int first = scores[0]; // 90 scores[2] = 100; // 修改元素 // 数组长度 int length = scores.length; // 5 // 遍历数组 for (int score : scores) { System.out.println(score); }

String str = "Hello, Java!"; // 常用方法 int len = str.length(); // 12 char ch = str.charAt(0); // 'H' String sub = str.substring(7); // "Java!" boolean contains = str.contains("Java"); // true String upper = str.toUpperCase(); // "HELLO, JAVA!" String replaced = str.replace("Java", "World"); // 字符串拼接 String name = "张三"; String greeting = "你好，" + name + "！"; // StringBuilder（高效拼接） StringBuilder sb = new StringBuilder(); sb.append("Hello"); sb.append(", "); sb.append("World"); String result = sb.toString();
