# JavaForge 🔥

> 前端工程师转型全栈工程师的学习之路 — 后端技术栈：Java

## 📖 项目简介

这是一个记录我从前端工程师转型为全栈工程师的学习仓库。通过系统化的学习计划和实践项目，目标在 **8 周内**掌握 Java 后端开发技能。

## 🎯 学习目标

- 掌握 Java 语言核心语法和面向对象编程
- 熟练使用 Spring Boot 框架开发 Web 应用
- 理解数据库设计和 RESTful API 设计
- 掌握前后端分离架构和全栈开发流程
- 具备独立开发和部署生产级应用的能力

## 📅 学习计划（8 周）

| 周次 | 主题 | 目标 |
|------|------|------|
| 第 1 周 | Java 基础与 Web 开发入门 | 创建 Spring Boot 项目并实现 CRUD |
| 第 2 周 | API 设计与数据库 | 设计数据库结构和 RESTful API |
| 第 3 周 | 身份验证与安全 | 实现 JWT 身份验证和授权 |
| 第 4 周 | 前端集成与全栈开发 | 完成前后端联调 |
| 第 5 周 | 测试与质量保证 | 编写单元测试和集成测试 |
| 第 6 周 | 部署与 DevOps | Docker 容器化和云服务部署 |
| 第 7-8 周 | 项目实战 | 独立开发并部署完整的 Web 应用 |

## 📌 学习原则

1. **项目驱动**：通过实际项目串联知识点
2. **先跑通再理解**：先让代码工作，再深入原理
3. **对比学习**：利用前端经验理解后端概念
4. **师徒式指导**：跟随 AI 导师系统化学习
